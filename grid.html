The Grid

We provide a mobile-first, fluid grid to help layout pages with ease. To start a new grid, create a div with the class of .thl-col-container. Then add columns inside it using .thl-col .thl-col-n-xs, where n = 5-100 in increments of 5, with the addition of 33 to let us do columns in thirds.

Important: Column classes should generally be used on container elements. It's bad practice to apply grid-based widths directly to elements such as form fields, headers or images. Also, avoid applying styles directly to a column element. Instead, nest your content within the column and apply styling there.

.thl-col .thl-col-30-xs
.thl-col .thl-col-20-xs
.thl-col .thl-col-50-xs
<div class="thl-col-container">
<div class="thl-col thl-col-30-xs border-mid-xs">.thl-col .thl-col-30-xs</div>
<div class="thl-col thl-col-20-xs border-mid-xs">.thl-col .thl-col-20-xs</div>
<div class="thl-col thl-col-50-xs border-mid-xs">.thl-col .thl-col-50-xs</div>
</div>

Grid Max-Width

By default, our grid will span 100% of the browser viewport. If you want to control that, add .thl-col-max next to your .thl-col-container class. This will limit the width of the grid to 1260px.

1260px max-width
<div class="thl-col-container thl-col-max">
<div class="thl-border-dark-xs p-2-xs">1260px max-width</div>
</div>

Gutters

The Leap grid doesn't include gutter by default. But there are times when you want to include gutters between your grid. For those times, we have .thl-col-gutters, which can be added next to .thl-col-container to automatically assign gutters to the internal columns. You may also choose to apply gutters manually (for a little more control) by using our padding utility classes next to the column classes.

Automatic Gutters

Gutters will not be applied to nested columns unless applied to its parent container.

.thl-col
.thl-col
.thl-col
.thl-col
<div class="thl-col-container thl-col-gutters">
<div class="thl-col thl-col-40-xs">
<div class="thl-border-mid-xs">.thl-col</div>
</div>
<div class="thl-col thl-col-60-xs">
<div class="thl-col-container">
<div class="thl-col thl-col-33-xs border-mid-xs">.thl-col</div>
<div class="thl-col thl-col-33-xs border-mid-xs">.thl-col</div>
<div class="thl-col thl-col-33-xs border-mid-xs">.thl-col</div>
</div>
</div>
</div>

Manual Gutters

With manual gutters, any styling should be applied to a nested element, not the columns themselves.

.thl-col
.thl-col
.thl-col
<div class="thl-col-container">
<div class="thl-col thl-col-30-xs p-r-1-xs">
<div class="thl-border-mid-xs">.thl-col</div>
</div>
<div class="thl-col thl-col-30-xs p-r-5-xs">
<div class="thl-border-mid-xs">.thl-col</div>
</div>
<div class="thl-col thl-col-40-xs">
<div class="thl-border-mid-xs">.thl-col</div>
</div>
</div>

Centering Columns

Sometimes you want to center a single column within its container. This is possible by adding .thl-col-center next to your .thl-col thl-col-n-xs class. You may also control column centering by using our margin and float utility classes. Simply apply .m-auto-xs .thl-float-none-xs and you'll get the same styles as .thl-col-center.

.thl-col .thl-col-50-xs .thl-col-center
.thl-col .thl-col-30-xs .thl-col-center
<div class="thl-col-container border-xs p-1-xs thl-m-b-1-xs">
<div class="thl-col thl-col-50-xs thl-col-center border-dark-xs">.thl-col .thl-col-50-xs .thl-col-center</div>
</div>
<div class="thl-col-container border-xs p-1-xs">
<div class="thl-col thl-col-30-xs thl-col-center border-dark-xs">.thl-col .thl-col-30-xs .thl-col-center</div>
</div>

Column Offsets

Move columns over by different grid widths by using .thl-col-offset-n-xs. Keep in mind that the columns and offsets used within a .thl-col-container shouldn't add up to more than 100.

.thl-col-40-xs
.thl-col-40-xs .thl-col-offset-30-xs
<div class="thl-col-container border-xs p-1-xs thl-m-b-1-xs">
<div class="thl-col thl-col-40-xs border-dark-xs">.thl-col-40-xs</div>
<div class="thl-col thl-col-40-xs thl-col-offset-20-xs border-dark-xs">.thl-col-40-xs .thl-col-offset-30-xs</div>
</div>

Responsive Breakpoint Classes

Our mobile first grid comes with a set of breakpoint class suffixes that can be used to override columns across breakpoints. For example, if a div needs to take up the full width of the page across all display sizes, use .thl-col-100-xs. If that same div needs to be half width at the medium breakpoint and a quarter at our large breakpoints, add .thl-col-50-md and .thl-col-33-lg. Resize the browser to see the effect in action. Note that there are also suffixes for small and extra large. You can learn more about the breakpoints in our responsive documentation
